---
title: "Environment Variables"
---

This page defines all of the Huxley configuration environment variables.

### General

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| HUXLEY\_LOG\_LEVEL | Text | info | Possible values: debug, info, warning, error, fatal |
| HUXLEY\_ENVIRONMENT | Text | production | Sets the application environment |
| HUXLEY\_ROLES | Text | api,scheduler,worker,reaper | Sets the roles that the process will operate as<br />comma separated string. Possible values: <br />api, scheduler, worker, reaper |

### UI

| Name | Type | Default | Description |
| :-- | :-- | :-- | :-- |
| HUXLEY\_URL | Text | localhost | Sets the Huxley URL (ex: example.com) |
| HUXLEY\_PROTOCOL | Text | http | Possible values: http, https |
| HUXLEY\_PORT | Number | 4267 | Sets the TCP port for Huxley to listen on |

### PostgreSQL

| Name | Type | Default | Description |
| :-- | :-- | :-- | :-- |
| HUXLEY\_DB\_HOST | Text | localhost | PostgreSQL instance hostname |
| HUXLEY\_DB\_PORT | Number | 5432 | PostgreSQL instance TCP port |
| HUXLEY\_DB\_USERNAME | Text |  | Username |
| HUXLEY\_DB\_PASSWORD | Text |  | Password |
| HUXLEY\_DB\_DATABASE | Text | Huxley | Database name |
| HUXLEY\_DB\_USE\_TLS | Boolean | True | Set to false to disable use of TLS for database connection |
| HUXLEY\_DB\_MIN\_CONNECTIONS | Number | 5 | Minimum number of database pool connections to keep alive |
| HUXLEY\_DB\_MAX\_CONNECTIONS | Number | 20 | Maximum number of database pool connections |
| HUXLEY\_DB\_IDLE\_TIMEOUT | Number | 600 | Seconds for idle database pool connection to be closed |
| HUXLEY\_DB\_POOL\_ACQUIRE\_TIMEOUT | Number | 5 | Seconds for database pool connection timeout |

### Redis/ValKey

| HUXLEY\_CACHE\_PORT | Number | 6379 | Redis/ValKey instance TCP port |
| HUXLEY\_CACHE\_USERNAME | Text |  | Username |
| HUXLEY\_CACHE\_PASSWORD | Text |  | Password |
| HUXLEY\_CACHE\_DB | Number | 0 | Database number |
| HUXLEY\_CACHE\_USE\_TLS | Boolean | True | Set to false to disable use of TLS for Redis connection |
| HUXLEY\_CACHE\_TIMEOUT | Number | 10 | Seconds for Redis connection timeout |
